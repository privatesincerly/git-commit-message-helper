### A small helper to write pretty commit messages (work in progress).

*gcmh* is a small script that helps developers save time by writing commit messages with emojis and prefixes.

#### How to use:
To use *gcmh*, simply place the script in your project folder (where your git repository lives). Run it each time you need to commit something, and then just use `git push`.

##### Fun fact: this repo's commits were pushed to this GitHub repository using *gcmh*. :)
